Creates a multi-dimensional array with the specified dimensions and with each element filled with the provided value.

example
type R = Array.Create<[Integer.FromDecimal<2>, Integer.FromDecimal<3>], 1>;
// => [[1, 1, 1], [1, 1, 1]]
