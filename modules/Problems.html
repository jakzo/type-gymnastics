NQueens

NQueens<ToPlace, GridSize>: _NQueens<ToPlace, GridSize>
  • Defined in problems/n-queens.d.ts:17

Solves the classical n queens problem.

See NQueensVisualized to get a more human-friendly output.

example
type R = Problems.NQueens<Integer.FromDecimal<8>, Integer.FromDecimal<8>>;
// => [["0b11", "0b111"], ["0b1", "0b011"], ["0b011", "0b101"], ["0b01", "0b001"], ["0b101", "0b11"], ["0b111", "0b01"], ["0b001", "0b1"], ["0b", "0b"]]

  • GridSize: Number = FromDecimal<8>

NQueensVisualized

NQueensVisualized<Queens, GridSize>: _NQueensVisualized<Queens, GridSize>
  • Defined in problems/n-queens.d.ts:37

Returns a visual representation of a result from NQueens. This string is designed to be viewed in the IntelliSense tooltip that appears when hovering the mouse over a type, and may not appear correctly in other locations.

example
type R = Problems.NQueensVisualized<
  Problems.NQueens<Integer.FromDecimal<4>, Integer.FromDecimal<4>>,
  Integer.FromDecimal<4>
>; // => "
// ◻️ ♕ ◻️ ◼️
// ◼️ ◻️ ◼️ ♕
// ♕ ◼️ ◻️ ◼️
// ◼️ ◻️ ♕ ◻️"
